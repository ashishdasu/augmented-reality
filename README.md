## Environment

- OS: macOS 16.4
- OpenCV 4.13.0
- Compiler: AppleClang 17.0.0

## Build Instructions

Requires OpenCV 4 and CMake 3.14+.

```bash
mkdir build && cd build
cmake ..
make
```

### Run calibration

```bash
./calibrate          # default camera
./calibrate 1        # use camera index 1 (e.g. if iPhone takes index 0)
```

Controls: `s` save frame, `c` run calibration (need 5+ frames), `w` write calibration.yml, `q` quit

### Run augmented reality

Requires `calibration.yml` in the build directory (generated by calibrate).
Also requires `car.obj` in the build directory.

```bash
./augmented_reality          # default camera
./augmented_reality 1        # use camera index 1
```

### Run Harris corner detection

```bash
./features
```

Use the Threshold and Block Size trackbars to adjust detection sensitivity.
